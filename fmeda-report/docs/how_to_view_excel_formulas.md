# 如何在 Excel 中查看隱藏公式 - 完整教學

## 概述

Excel 中的公式並不是「隱藏」的，而是**預設不顯示**。當您打開 Excel 時，看到的是公式的**計算結果**，而不是公式本身。本文教您多種方法來查看和分析 Excel 中的公式。

---

## 方法 1：在儲存格中直接查看公式（最簡單）

### 步驟
1. **點擊任意包含公式的儲存格**
2. **查看「公式欄」**（在工具欄下方，顯示 `fx` 符號的位置）

### 示例
如果您點擊 BlockList!C2，公式欄會顯示：
```
=IF($AJ$2="No",FailureRateCalc!E23,(FailureRateCalc!E23+FailureRateCalcIC!$D$129*(BlockList!E2/SUM(BlockList!$E$2:$E$7))+FailureRateCalcIC!$D$130*(BlockList!E2/SUM(BlockList!$E$2:$E$7))))
```

### 優點
- 簡單直觀
- 可以直接編輯公式
- 可以看到公式引用的儲存格會被高亮顯示

---

## 方法 2：顯示整個工作表的公式（全局查看）

### 步驟
1. 按下 **Ctrl + `**（Ctrl + 反引號/重音符，通常在 Esc 下方）
2. 整個工作表的公式會全部顯示（而不是計算結果）
3. 再按一次 **Ctrl + `** 恢復顯示結果

### 優點
- 一次看到所有公式
- 便於比較不同儲存格的公式
- 可以看到公式的完整邏輯結構

### 缺點
- 列寬可能不足，公式被截斷
- 難以看到公式的完整內容

### 解決方案
按下 **Ctrl + `** 後，可以調整列寬來看完整公式。

---

## 方法 3：追蹤公式的依賴關係（進階）

### 步驟
1. 點擊包含公式的儲存格
2. 進入「公式」選項卡
3. 點擊「追蹤從屬項」或「追蹤先行項」

### 功能
- **追蹤先行項**：顯示這個公式引用的所有儲存格
- **追蹤從屬項**：顯示哪些公式引用了這個儲存格

### 優點
- 可以視覺化公式的依賴關係
- 便於理解複雜的計算鏈

---

## 方法 4：使用「求解器」或「數據表」分析（進階）

### 步驟
1. 進入「數據」選項卡
2. 點擊「假設分析」→「數據表」或「求解器」

### 用途
- 理解公式如何隨輸入變化
- 找出公式的敏感性參數

---

## 方法 5：使用 Python 腳本批量提取公式（本項目使用的方法）

### 原理
使用 `openpyxl` 庫讀取 Excel 檔案，直接訪問每個儲存格的 `.value` 屬性（當儲存格包含公式時）。

### 代碼示例
```python
import openpyxl

wb = openpyxl.load_workbook('RD-03-008-01FMEDAReport.xlsx', data_only=False)
ws = wb['BlockList']

for row in ws.iter_rows():
    for cell in row:
        if cell.value and isinstance(cell.value, str) and cell.value.startswith('='):
            print(f"{cell.coordinate}: {cell.value}")
```

### 優點
- 可以批量提取所有公式
- 可以進行語義分析和去重
- 可以生成公式依賴圖譜

### 缺點
- 需要編程知識
- 需要安裝 Python 和相關庫

## 方法 6：使用 Excel 內置的「名稱管理器」（進階）

### 步驟
1. 進入「公式」選項卡
2. 點擊「名稱管理器」
3. 查看所有命名範圍和公式

### 用途
- 查看 Excel 中定義的所有命名公式
- 理解複雜的命名約定

## 本項目的發現

### FMEDA 公式統計
- **總公式數**：567,973 個
- **邏輯上不同的公式**：827 個
- **平均重複次數**：686.8 倍

這說明 FMEDA 的大部分公式都是**模板化的**，即相同的邏輯被應用到不同的行。

### 公式分類
| 類型 | 數量 | 說明 |
|:-----|:----:|:-----|
| CONDITIONAL | 374 | IF 條件判斷 |
| LOOKUP | 數百 | VLOOKUP、HLOOKUP 等查表公式 |
| AGGREGATE | 數十 | SUM、AVERAGE、COUNT 等聚合公式 |
| MATH | 數十 | 基本數學運算 |
| EXPONENTIAL | 數個 | EXP 指數函數 |

### DFA vs FMEDA 的共同公式
兩份報告共享 **14 個邏輯上相同的公式**，包括：
- `=SUM(D_ROW:D_ROW)` — 求和
- `=B_ROW*H_ROW` — 乘法
- `=IF(B_ROW<45, 5, IF(B_ROW<70, 2, IF(B_ROW<120, 1, 0.5)))` — 分級條件判斷

這反映了 DFA 和 FMEDA 在**計算邏輯上的高度一致性**。

## 實用技巧

### 1. 快速定位複雜公式
如果公式很長，使用「公式欄」中的「編輯」功能，可以看到公式中各個部分的顏色編碼。

### 2. 複製公式用於分析
在 Excel 中複製公式，粘貼到文本編輯器中，便於分析和註解。

### 3. 使用「追蹤」功能找出錯誤
如果計算結果不對，使用「追蹤先行項」找出所有依賴的儲存格，逐一檢查。

### 4. 使用「監視窗口」監控多個公式
在「公式」選項卡中，點擊「監視窗口」，可以同時監控多個儲存格的值和公式。
