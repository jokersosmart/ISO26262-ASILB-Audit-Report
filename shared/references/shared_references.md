# 共用參考文件解讀

> 本文件整理 DFA 和 FMEDA 兩份報告共同引用的國際標準，說明每份標準在功能安全認證中的角色。

---

## 1. ISO 26262:2018 — 道路車輛功能安全

### Part 5：硬體層級產品開發
- **角色**：定義 SPFM、LFM、PMHF 三個安全度量的目標值（Table 6/7/8）
- **DFA 引用**：作為 CCF β 因子計算結果的最終驗證標準
- **FMEDA 引用**：作為 FMEDA 主表故障分類和安全度量計算的判定標準
- **ASIL B 目標值**：SPFM ≥ 90%, LFM ≥ 60%, PMHF < 10⁻⁷/h

### Part 9：ASIL 導向與安全導向分析
- **角色**：第 7 章定義了相依故障分析（DFA）的完整方法論
- **DFA 引用**：整份 DFA 報告的法規基礎，ISO 26262-9 CH.7 sheet 直接收錄原文
- **FMEDA 引用**：CCF 分析方法的法規依據

### Part 11：半導體應用指引
- **角色**：§4.7 專門針對半導體 IC 的 DFA 方法提供指引
- **DFA 引用**：DFI examples sheet 的 7 類 DFI 範例來源
- **FMEDA 引用**：半導體特有失效模式的定義來源

---

## 2. IEC 61508 Ed.2 — 功能安全

### Annex D：共因故障的定量評估
- **角色**：提供 β 因子計算的標準化評分方法
- **DFA 引用**：CCF Calc. On-Chip Red. 和 CCF Calc. Logic 的計算方法來源
- **FMEDA 引用**：同上，且 FMEDA 的 βIC = 47%（啟用增加因子）

### Table D.1 / D.2：β 因子查表
- **DFA 引用**：Logic sheet 使用 Table D.2 的 Z 值查表
- **FMEDA 引用**：同上

---

## 3. IEC TR 62380 — 電子元件可靠度數據

- **角色**：提供 IC 失效率預測的數學模型
- **DFA 引用**：未直接使用（DFA 不計算失效率）
- **FMEDA 引用**：FailureRateCalc sheet 的核心計算模型，公式 λ = f(N_transistors, years, Q_factor)

---

## 4. IEC 61709 — 電子元件可靠度參考條件

- **角色**：定義元件在不同環境條件下的失效率修正因子
- **DFA 引用**：作為環境因子的參考
- **FMEDA 引用**：FailureRateCalc 中的品質因子和環境因子來源

---

## 5. SN 29500 — Siemens 失效率手冊

- **角色**：提供被動元件（電阻、電容等）的失效率數據
- **DFA 引用**：未直接使用
- **FMEDA 引用**：BlockList 中被動元件失效率的數據來源

---

## 交叉引用矩陣

| 參考標準 | DFA 使用 | FMEDA 使用 | 共用程度 |
|:---------|:---------|:-----------|:---------|
| ISO 26262-5 | 安全度量目標值 | 安全度量目標值 + 故障分類 | 高 |
| ISO 26262-9 CH.7 | 法規基礎（直接收錄） | CCF 方法論 | 高 |
| ISO 26262-11 §4.7 | DFI 範例來源 | 半導體失效模式 | 中 |
| IEC 61508 Annex D | β 因子計算 | β 因子計算 | 高 |
| IEC TR 62380 | 未使用 | 失效率預測核心 | 低 |
| IEC 61709 | 環境因子參考 | 品質/環境因子 | 中 |
| SN 29500 | 未使用 | 被動元件失效率 | 低 |
